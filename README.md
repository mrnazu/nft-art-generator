## Getting Started:
**Clone the repository:**
   ```bash
   git clone https://github.com/mrnazu/nft-art-generator.git
   ```
**Install dependencies (if any):**
   `npm install`
**Generate your image:**
    `node .\scripts\generateRarityConfig.js`
    `node .\scripts\create.js`
